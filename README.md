#Hash Analyzer

This is a simple HTML-based tool that allows you to identify the type of hash for a given input.


####Currently supported hashing algorithms:
-SHA1
-SHA256
-SHA512
-MD5
-Blowfish

_____

>Note: This tool uses Bootstrap and other third party libraries.
